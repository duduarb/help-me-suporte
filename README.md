Site feito para me auxiliar no meu trabalho de analista de supote técnico. Como realizo o suporte de diversos produtos resolvido criar um local aonde consigo informações rápidas para os atendimentos via chat e ligação.

Foi criado algo básico utilizando html, css e js para estrutura do site e o banco de dados é utilizado o supabase.
A ideia do site é você poder adicionar, editar e remover conteúdos através do próprio site. Como não sei quase nada sobre criar um login para o site no momento e ao pesquisar achei um pouco complexo, resolvi apenas colocar a função que ao adicionar/editar/excluir algum conteúdo, vai pedir uma senha.
